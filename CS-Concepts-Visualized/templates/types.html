Types

As we have not only numbers but we also have alphabets or characters, we need to differentiate between them and let the computer know how much data we want each type to store and what kind of data, so as to ensure correctness and optimize memory usage (mostly only needed in simpler programming langauges like C).

What are Types?

In computer science, types define the kind of data a variable can hold and how the operations on that data are performed. In C, types are broadly categorized into basic types (like integers and floating-point numbers), derived types (arrays, pointers, etc.), and user-defined types (structs, enums).

Example code of common Types


                #include <stdio.h>

                    // define a struct
                    struct Person
                    {
                        char name[50];
                        int age;
                    };

                    // define an enum
                    enum Day { Monday, Tuesday, Wednesday, Thursday, Friday, Saturday, Sunday };

                    int main()
                    {
                        // basic types
                        int count = 10;
                        float price = 19.99;
                        double preciseValue = 3.14159265359;
                        char initial = 'A';

                        // array
                        int numbers[] = {1, 2, 3, 4, 5};

                        // struct (referencing the struct data type)
                        struct Person person1 = {"Alice", 25};

                        // enum
                        enum Day today = Wednesday;

                        // output examples
                        printf("Integer (count): %d\n", count);
                        printf("Float (price): %.2f\n", price);
                        printf("Double (precise value): %.10f\n", preciseValue);
                        printf("Character (initial): %c\n", initial);

                        // more elaborate output examples
                        printf("Array (first number): %d\n", numbers[0]);
                        printf("Struct (person's name and age): %s, %d\n", person1.name, person1.age);
                        printf("Enum (today): %d (Wednesday is 2 in enum)\n", today);

                        return 0;
                    }

Different Types in C

Type Definition Use Case Examples
int represents integers (whole numbers), typically 4 bytes in size. counting items, looping, arithmetic operations. keeping track of counters, loop variables, IDs.
float represents single precision floating-point numbers (decimals). storing real numbers with limited precision. calculating areas, speed, or any physical quantities.
double represents double-precision floating-point numbers. calculations requiring high precision, e.g., physics. similar to float but a double is 64 bits vs float which is 32 bits and as a result can represent much larger numbers.
char Represents a single character, typically 1 byte in size. storing characters, managing text as arrays of char. storing individual characters or strings using arrays of char.
void represents "no type," often used for functions that do not return a value. declaring functions like void main(). function declaration with a NULL return value int main(void)
Array a collection of elements of the same type stored in contiguous memory. storing lists, matrices, or strings. storing and manipulating collections of same data types like sensor number of days in a year.
Struct a collection of different types grouped together as a single entity. representing complex objects, like a person. representing more complex entities like a "book" with a title, author, and price.
Enum represents a set of named integer constants. defining states, modes, or options. using symbolic names for constant values, eg, traffic light states (Red, Yellow, Green).

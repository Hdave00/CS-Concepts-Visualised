Algorithms

Algorithms are the foundation for problem-solving in computer science/programming, and are used in a variety of fields, including data processing, artificial intelligence, cryptography, and more.

What are algorithms?

Algorithms are step-by-step procedures or instructions for solving problems. In computer science, they are used to perform calculations, manipulate data, and automate reasoning tasks. The example below demonstrates the Bubble Sort algorithm, which is a simple sorting technique. It repeatedly steps through the list, compares adjacent elements, and swaps them if they are in the wrong order. This process continues until the list is fully sorted.

Let's take an example of a simple sorting algorithm called Bubble Sort. Bubble Sort algorithm is simple to understand and easy to implement, making it a good starting point for understanding sorting concepts. It essentially, checks the "left" and "right" or adjascent data in an array in memory, and swaps them as it travrses the data set until the conditions of sorting are met. These conditions are usually to arrange/sort the data in ascending or descending. It's time complexity is O(nÂ²) in the average and worst cases, meaning it performs best on small or nearly sorted arrays. Example code of an algorithm that is using bubble sort


                #include <stdio.h>
                    // our sorting algorithm
                    void bubbleSort(int arr[], int n)
                    {
                        for (int i = 0; i < n - 1; i++)
                        {
                            for (int j = 0; j < n - i - 1; j++)
                            {
                                if (arr[j] > arr[j + 1])
                                {
                                    // Swap elements
                                    int temp = arr[j];
                                    arr[j] = arr[j + 1];
                                    arr[j + 1] = temp;
                                }
                            }
                        }
                    }
                    // main function that allows us to print those numbers, now sorted from the array
                    int main()
                    {
                        int arr[] = {5, 2, 9, 1, 5, 6};
                        int n = sizeof(arr) / sizeof(arr[0]);

                        bubbleSort(arr, n);

                        printf("Sorted array: ");
                        for (int i = 0; i < n; i++)
                        {
                            printf("%d ", arr[i]);
                        }
                        printf("\n");

                        return 0;
                    }

Use Cases of Algorithms

In our very large world, we have very large data sets, we all have devices, maybe 2-3 per person but large companies often have machines numbering in hundreds of thousands, each of which have their own data. Algorithms help automate and make those very large data sets in this context, faster to index specific data in that whole set, displaying ordered lists (e.g., search engine results, e-commerce product listings).

Algorithms also play a key role in finding the shortest or optimal path between two points. This applies not only to GPS Navigation systems, but also to more complex, predetermined and controlled envirnments like character movement and behaviour in video games by use of AI and even in our internet for the most optimal packet routing!

Search engines and recommendation systems rely on complex algorithms to deliver relevant results and personalized suggestions. We all would have experienced this wih Google searchs or Netflix/Youtube reccomendations.

Algorithms are essential in cybersecurity for encrypting data, ensuring privacy, and securing communications against attacks. We also know about recent surge in AI based application computing. The foundation for these AI aplications like autonomous vehicles, image and speech recognition are built on algorithms, unbelievably complex, but still the same principles.

Algorithms not only help in powering our massive audio/video streaming itself, but when these huge service providers/companies stream 4k resolution data or for more complex applications like data transmission over networks, there is no guarantee that the the reciever has the same resources for data storage and consumption, so data compression whether its in ZIP files or MP3/MP4 or FLAC audio files is also handled by algorithms

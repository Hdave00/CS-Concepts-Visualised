Commands

There are a few commands which can be helpful and useful in a pinch, to interact with IDEs and manipulate files and code. Go to the example below for an idea of some of the essential commands.

Comments

Comments in code/programming are essential lines of explanation in plain english (or whatever other languages you speak) for explaining the logic of the code or specific links that you have included or for keeping track of documentation and your thought process. Its a very good idea idea to include comments not only to visualize your thought process, but also for people who will be reading your code as everyone has different styles of writing code or you might be working together in a team so you have to make your code be intuitive to read for others (usually helpful, rarely implemented...).

Here are a few examples of linux terminal commands


                # List files in the current directory
                ls

                # Change directory
                cd /path/to/directory

                # Display the contents of a file
                cat filename.txt

                # Unzip a zip file
                unzip cat.zip

                # Changing directory within your folder (from a file in your main folder to your main folder)
                cd ..

                # Copy a file
                cp source.txt destination.txt

                # Move a file
                mv oldname.txt newname.txt

                # Remove a file
                rm filename.txt

                // including the base library

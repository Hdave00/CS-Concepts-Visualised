Arrays

Arrays are a data structure in memory that are essential for organising and maintaining bits of information.

What are Arrays?

Arrays are a collection of elements, all of the same type, stored at contiguous memory locations(end to end, in order). They allow efficient access to data and manipulation of data by using an index to refer to specific element(s).

Example code of an array


                // including the base library
                #include <stdio.h>

                int main(void) {
                    // declaring the array
                    int numbers[5] = {10, 20, 30, 40, 50};

                    for (int i = 0; i < 5; i++) {
                        printf("%d\n", numbers[i]);
                    }
                    return 0;
                }

Use Cases for Arrays

  • Storing and manipulating collections of data, like a list of scores.
  • Implementing data structures like stacks, queues, and matrices.
  • Iterating over data for counting, keeping track of the count, and updating the value of the variable.
  • Efficiently accessing and updating elements using indices.
  • Manipulating bits of data for arranging them in a specified order, like for encryption in cybersecurity.
  • The Caesar Cipher is an infamous example of "encrypting" text with a "key". This time with bits in an array instead of a paper with a wax seal!

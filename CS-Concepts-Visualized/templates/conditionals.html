Conditionals

In C, conditionals are used to execute specific blocks of code based on the evaluation of boolean expressions. It's similar to saying, "IF a number is GREATER than 0, then it is POSITIVE, ELSE the number is NEGATIVE." simple enough right?

What are Conditionals?

Conditionals are essentially statements in code that check the condition of what that statement contains, in the below examples you will see that by understanding and using these conditionals, you can control the flow of a C program (or any other language but let's not go down that rabbit hole) effectively!

Example code of using conditionals to check if a number is positive, negative, or zero.


                // including the base library
                #include <stdio.h>

                int main(void)
                {
                    // declaring a number variable
                    int number;

                    // asking the user for a number
                    printf("Enter a number: ");
                    scanf("%d", &number);

                    // checking if the number is positive, negative, or zero
                    if (number > 0)
                    {
                        printf("The number is positive.\n");
                    }
                    else if (number < 0)
                    {
                        printf("The number is negative.\n");

                    }
                    else
                    {
                        printf("The number is zero.\n");
                    }
                    return 0;
                }
